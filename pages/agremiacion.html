Agremiación Odontológica - Entrega de Bonos

WebApp de registro y entrega de bonos para consultorios odontológicos

Información Rápida

Nombre del proyecto: Agremiación Odontológica - Entrega de Bonos

Descripción: Este proyecto es una web que facilita la alta y entrega de bonos odontológicos. Fue desarrollado como trabajo de campo para la cátedra de Diseño de Sistemas en la UTN FRLP

Tipo: Proyecto académico - Cátedra Diseño de Sistemas UTN FRLP

Rol: Desarrollador Full-Stack

Fecha: Febrero 2024 - Marzo 2024

Colaboradores: Juan Manuel Semper

Ver código

Desafío

El proyecto consistió en desarrollar un Sistema de Carga y Entrega de Bonos para la Agremiación Odontológica, destinado a ser utilizado en consultorios. El desafío principal fue digitalizar y automatizar el proceso de gestión de bonos, el cual es fundamental para la facturación y el servicio a los profesionales. La solución debía facilitar tanto el alta de bonos en el sistema como su posterior asignación controlada a los odontólogos.

Enfoque y Proceso

Enfoque se centró en crear un flujo de trabajo sencillo y seguro para la manipulación de información sensible (datos de odontólogos y obras sociales). Prioricé tres áreas clave:

  • Registro de Entidades: Implementación de módulos para cargar la información personal de los odontólogos.
  • Gestión Numérica de Bonos: Creación de una interfaz que permita dar de alta lotes de bonos y asignarlos por rangos numéricos.
  • Detalle de Asignación: Asegurando que cada entrega de bonos quede registrada con precisión, detallando la obra social y el intervalo exacto (desde/hasta) del talonario asignado.

Se utilizó un enfoque de desarrollo centrado en la confiabilidad de los datos.

Solución Implementada

Construí una Web App de Gestión que sirve como columna vertebral para el manejo de bonos. La solución centraliza dos procesos críticos:

  • Carga de Odontólogos: Permite dar de alta y gestionar a los profesionales con todos sus datos personales.
  • Entrega de Bonos: La funcionalidad principal que permite dar de alta nuevos bonos y entregarlos, asignándolos a odontólogos específicos. Esta asignación requiere detallar la Obra Social correspondiente y el rango de números de bono exacto que se está entregando (por ejemplo, del 5001 al 6000).

Características Principales

Gestión de Odontólogos

Módulo completo para el alta, baja y modificación de profesionales, almacenando datos clave para la facturación.

Control de Stock de Bonos

Registro y seguimiento de los talonarios de bonos disponibles, permitiendo la carga masiva por rangos numéricos.

Asignación Trazable

Vinculación segura entre bonos, odontólogos y obras sociales, asegurando la trazabilidad de cada bono entregado.
